Примеры использования JavaScript функций

Добавление продуктов

Добавить продукт в дверцу холодильника:
addProduct('fridge-door-items', {
    name: 'Молоко',
    quantity: '1 л',
    icon: 'fas fa-wine-bottle'
});
Добавить продукт в заморозку:
addProduct('fridge-freezer-items', {
    name: 'Свинина',
    quantity: '2 кг',
    icon: 'fas fa-drumstick-bite'
});
Добавить продукт на верхнюю полку шкафа:
addProduct('cabinet-top-items', {
    name: 'Чай',
    quantity: '100 г',
    icon: 'fas fa-coffee'
});
Добавить лекарство в 1 аптечку:
addProduct('pharmacy-kit1-items', {
    name: 'Аспирин',
    quantity: '30 таблеток',
    icon: 'fas fa-pills'
});

Управление данными

Обновить дату:
updateLastUpdated('20.12.2024');
Поиск продуктов:
searchProducts('молоко');
Очистить поиск:
clearSearch();
Экспорт данных:
exportData();
